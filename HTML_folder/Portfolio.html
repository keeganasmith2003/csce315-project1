My Software Projects

College Football Schedules

In this project I created most of the backend funcionality for the College Football Schedules app which is available on iOS - College Football Schedules '23


I helped configure the cloudkit database and automated the process of pulling data from the cfbd api, transforming the data, and inserting it into the database. This was done using javascript and the cloudkit JS library. I also created the UI and functionality for the favorites page which allows users to store and view their favorite teams.


The app currently has a little over 1,000 subscribers (many more downloads). We expect this number to grow as the college football season progresses.
